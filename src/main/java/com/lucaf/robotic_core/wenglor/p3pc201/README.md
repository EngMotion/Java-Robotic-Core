# How to configure WENGLOR on the web interface

1. Open the sensor configuration page in your web browser.
2. Port Functions - Pin 4 - Set function to IO-Link autostart
3. IODD Assignment - Should be autoselected if the correct IODD file is loaded.
4. To upload the IODD file, to the Master configuration page, select "Upload IODD file" and select the correct IODD file for the WENGLOR P3PC201 sensor.
